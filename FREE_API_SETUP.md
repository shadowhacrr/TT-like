## 🛠️ Troubleshooting

### Error: "TikTok-Api not installed"
**Solution:**
```bash
pip install TikTok-Api
playwright install
```

### Error: "No module named 'TikTokApi'"
**Solution:**
```bash
pip uninstall TikTok-Api
pip install TikTok-Api
```

### Error: "Playwright not found"
**Solution:**
```bash
playwright install chromium
```

### Error: "API rate limited"
**Solution:**
- Thodi der wait karo (5 min)
- SociaVault add karo (backup)
