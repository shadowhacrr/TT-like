## 🚀 10-Minute Setup

### Step 1: Download & Extract
Download ZIP, extract to folder

### Step 2: Install Dependencies
```bash
pip install -r requirements.txt
pip install TikTok-Api
playwright install
```

### Step 3: Configure
Edit `config.py`:
```python
BOT_TOKEN = "your_token_from_@BotFather"
```

### Step 4: Run
